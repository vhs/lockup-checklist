# lockup-checklist

A checklist for locking up VHS.
This is a static HTML page that contains:
* Checkboxes for each lockup item
* A progress bar
* A banner when you've completed everything
* A reset button

Checklists where you need to actually physically mark items as done have been shown to be more effective than simple 'mental' or 'dot point' checklists, especially in safety critical applications like aviation and medicine.
Locking up VHS feels like our equivalent of a safety critical operation, so a checklist where you can check each item off feels appropriate.

It's deliberately very barebones, and attempts to follow some checklist best practices including:
* Listing things that can be done together, together. For us this means listing things that are in close physical proximity (i.e. same room) nearby
* Simple summaries at the start of each item
* Reminders of only the most important steps. This checklist is not a replacement for an understanding of the process for beginners, it's a tool to aid people who already understand the process to ensure it is completed accurately.
* Explict "read-do" pattern. You read the item, then do it and check it off
